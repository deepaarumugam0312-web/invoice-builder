# Invoice Builder

React + Tailwind invoice generator with PDF export.

Getting started

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Build

```bash
npm run build
```

Notes

- Exports PDF using `html2canvas` + `jsPDF`.
- UI uses TailwindCSS; Tailwind is configured in `tailwind.config.cjs`.
